Quick Start Guide

Get your first payment working in 4 steps.

Step 1: Get your API key

Sign up at xcoingateway.com and navigate to Settings → API Keys. Generate a new key and note the secret — you'll need both.

Step 2: Install the SDK

Terminal
# Node.js
npm install @xcoingateway/sdk

# Python
pip install xcoingateway

# PHP
composer require xcoingateway/sdk

Step 3: Create a payment

Use your API key to create a payment. The response includes a payment URL you can redirect your customer to.

Step 4: Handle the webhook

When the payment is confirmed, we'll send a signed webhook to your callback URL. Verify the HMAC signature and update your order status.

JavaScript
import { XCoinGateway } from '@xcoingateway/sdk';

const xcg = new XCoinGateway({
  apiKey: 'xcg_live_...',
  secret: 'xcg_secret_...'
});

// Create a payment
const payment = await xcg.createPayment({
  price_amount: 99.99,
  price_currency: 'USD',
  order_id: 'ORD-123',
  ipn_callback_url:
    'https://you.com/webhook'
});

// Redirect customer
res.redirect(payment.payment_url);
Webhook Handler
app.post('/webhook', (req, res) => {
  const isValid = xcg.verifyWebhook(
    req.body,
    req.headers['x-xcg-signature']
  );

  if (isValid) {
    // Update order status
    updateOrder(req.body.order_id,
      req.body.status);
  }
  res.status(200).send('OK');
});

Authentication

All API requests require authentication via your API key sent in the request header.

HTTP Headers
x-api-key: xcg_live_7f3a8b2c...
x-api-nonce: 1711843200
x-api-signature: HMAC-SHA256(body + nonce, secret)
Content-Type: application/json

Every request must include a nonce (Unix timestamp) and HMAC-SHA256 signature for security. Requests with invalid or expired signatures are rejected.
