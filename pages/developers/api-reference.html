API Reference

Base URL: https://api.xcoingateway.com/v1

Payments

Create, retrieve, and manage crypto payments.

MethodEndpointDescription
POST/paymentsCreate a new payment
GET/payments/:idGet payment by ID
GET/paymentsList all payments (paginated)
GET/payments/:id/statusGet real-time payment status

Create Payment — Request

POST /payments
{
  "price_amount": 150.00,
  "price_currency": "USD",
  "pay_currency": "ETH",
  "order_id": "ORD-12345",
  "order_description": "Premium Plan",
  "ipn_callback_url": "https://you.com/hook",
  "success_url": "https://you.com/success",
  "cancel_url": "https://you.com/cancel"
}
