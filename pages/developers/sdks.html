Official SDKs

Client libraries for the most popular programming languages. Install, configure, and start accepting payments.

🟢

Node.js

Official JavaScript/TypeScript SDK with full type support.

npm i @xcoingateway/sdk
🐍

Python

Python 3.8+ SDK with async support and type hints.

pip install xcoingateway
🐘

PHP

PHP 8.0+ SDK compatible with Laravel and Symfony.

composer require xcoingateway/sdk
🔵

Go

Idiomatic Go client with context support and retries.

go get xcoingateway.com/sdk
💎

Ruby

Ruby 3.0+ gem with Rails integration helpers.

gem install xcoingateway
🌐

REST API

Use any language with our standard REST API and HMAC auth.

api.xcoingateway.com/v1

Quick Examples

Python
from xcoingateway import XCoinGateway

xcg = XCoinGateway(
    api_key="xcg_live_...",
    secret="xcg_secret_..."
)

payment = xcg.create_payment(
    price_amount=49.99,
    price_currency="USD",
    pay_currency="BTC",
    order_id="ORD-456"
)

print(payment.payment_url)
PHP
use XCoinGateway\Client;

$xcg = new Client(
    'xcg_live_...',
    'xcg_secret_...'
);

$payment = $xcg->createPayment([
    'price_amount' => 49.99,
    'price_currency' => 'USD',
    'pay_currency' => 'BTC',
    'order_id' => 'ORD-456',
]);

header("Location: ".$payment->payment_url);
